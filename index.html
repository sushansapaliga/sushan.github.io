The Taste

A Online portal for users to directly order dishes from the restaurant, The Taste.

Built using ReactJS as PWA and monolithic back-end application with Java Spring. We are using MongoDB as our Primary Database. For payment we are using Paytm Payment Gateway, to get customers current location and area name, we are using Google Maps API and for authentication we are using Firebase Authentication. The applications are hosted on Firebase Hosting and AWS Elastic Beanstalk.
